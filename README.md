# IAKOA — Résumé de l'application

***IAKOA est une application web conçue pour aider les utilisateurs à découvrir, créer et gérer des événements locaux de manière simple et visuelle. Elle cible les personnes souhaitant trouver des sorties, partager des rencontres ou organiser des événements communautaires dans leur ville.***

## Technologies utilisées

- ***Backend*** : TypeScript, Node.js avec NestJS — architecture modulaire, controllers/services, injection de dépendances. Authentification avec bcrypt (hachage des mots de passe) et tokens (JWT) pour la gestion des sessions.

- ***Frontend*** : TypeScript, React avec Next.js — rendu côté serveur / génération statique, routage et pages. Styles et utilitaires UI avec Tailwind CSS.

- ***Base de données*** : PostgreSQL géré via Prisma ORM — schéma typé, migrations, et requêtes type-safe.
