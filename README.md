## Technologies utilisées

- **Backend**
  - TypeScript
  - Node.js avec NestJS
  - Prisma (PostgreSQL)
  - JWT pour l'authentification
  - Bcrypt pour le chiffrement et la sécurité

Le backend est conçu pour une gestion centralisée des données, permettant un futur déploiement fluide sur des plateformes web Next.js , iOS en Swift et Android en Kotlin.
